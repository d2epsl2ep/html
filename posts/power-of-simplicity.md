黑白配色从不过时。它剥离了所有干扰，只留下本质。在界面设计中，少即是多——减少颜色、减少装饰，让内容和功能成为真正的主角。

## 为什么简约

我们常常陷入一个误区：认为更多的功能、更多的视觉效果等于更好的产品。但事实上，每一个多余的元素都在消耗用户的注意力。

简约不是简单，而是**精心选择的克制**。

> perfection is achieved not when there is nothing more to add, but when there is nothing left to take away. — Antoine de Saint-Exupéry

## 黑白设计的优势

1. **永恒的优雅** — 黑色和白色永远不会过时
2. **减少认知负担** — 用户不需要解码颜色的含义
3. **内容优先** — 没有色彩竞争，读者自然关注文字
4. **无障碍友好** — 高对比度易于阅读，对视觉障碍用户更友好

## 代码中的简约

简约同样适用于代码。好的代码应该是自解释的，不需要大量注释。它像好的 prose 一样，读起来顺畅自然。

```python
def fibonacci(n):
    a, b = 0, 1
    for _ in range(n):
        yield a
        a, b = b, a + b
```

简洁、清晰、高效。这就是代码中的黑白美学。
